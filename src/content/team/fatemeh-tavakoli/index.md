projects:
  - title: "Federated Learning on Non-Disjoint Graphs"
    description: "Federated learning is widely used for learning a global model when data is distributed across clients. Most prior work assumes independent data points, but there is often an underlying graph structuring the data, as seen in social networks, content recommendations, healthcare records, and bank transactions. Standard federated learning frameworks are not designed for graph data and cannot exploit graph structure for node classification. This work considers federated learning on graph data where a global graph is split among clients with a non-disjoint split, meaning some anchor nodes appear at multiple clients. We propose a new federated learning algorithm that leverages anchor nodes to augment local graph structure, improving node classification in a semi-supervised setting. Experiments on several graph datasets show our method outperforms standard baselines."
